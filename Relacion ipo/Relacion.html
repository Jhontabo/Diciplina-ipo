Relacion Ingenieria de software con IHM

La ingeniería de software está relacionada con lo que se conoce como IHM ya que los profesionales de esta carrera son capaces de escribir líneas de código para crear programas o aplicaciones las cuales hacen nuestro vida más fácil, estos programas o aplicaciones son las que se encargan de transformar las lineas de codigo de un lenguaje de programación a lenguaje máquina el cual las máquinas pueden leer y procesar para realizar acciones.

Todo internet y nuestro mundo actual está construido gracias a la programación por eso es que es una de las ramas más importantes dentro de la IPO ya que es la que nos ayuda a comunicarnos y darle órdenes a las máquinas para que realicen determinadas tareas y así hacer nuestra vida mucho más fácil y productiva
