El desarrollo de software se refiere a un conjunto
actividades informáticas dedicadas al proceso de
creación, diseño, despliegue y compatibilidad de software.

El software en sí es el conjunto de instrucciones o programas
que le dicen a una computadora qué hacer.
Es independiente del hardware y hace que las computadoras sean programables
